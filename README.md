## Description

This is common shareable eslint config.

## Dependency

You have to contain module as follows into your repository.

```
"eslint": "^8.0.0",
"prettier": "^2.0.0",
"react": "^17.0.0",
"typescript": "^4.0.0"

```

## How to use

### install

```
yarn add --dev eslint-config-tenryu
```

### into your eslint config, add extends keyword

```
{
  "extends": ["tenryu"],
}
```
